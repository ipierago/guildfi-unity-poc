# guildfi-unity-poc

## Setup

yarn

## To Build Unity

Open unity/guildfi-unity-poc-alpha in Unity and build to WebGL to public/unity/guildfi-unity-poc-alpha.

## To debug

yarn dev

## To Build

yarn build

## To Deploy

firebase deploy
